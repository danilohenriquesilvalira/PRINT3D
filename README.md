# PRINT3D — Peças e Produtos Impressos em 3D

[![Live Demo](https://img.shields.io/badge/Demo-Live-green?style=for-the-badge)](https://github.com/danilohenriquesilvalira/PRINT3D)

O **PRINT3D** é uma plataforma de e-commerce moderna e de alta performance dedicada à venda de produtos impressos em 3D, com foco em precisão, qualidade e design inovador (baseado na estética industrial premium).

## 🚀 Tecnologias Utilizadas

Este projeto foi construído com as tecnologias mais recentes do ecossistema de desenvolvimento web:

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router & Turbo Mode)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Componentes UI**: [ShadCN UI](https://ui.shadcn.com/) & Radix UI
- **Estado**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **Ícones**: [Lucide React](https://lucide.dev/)

## ✨ Funcionalidades Principais

- **Design Premium & Ultra-Responsivo**: Interface adaptada para desktop e mobile, com um layout de 2 cards por linha no telemóvel para melhor visibilidade.
- **Lupa de Zoom Interativa**: Sistema de inspeção de detalhes nos produtos, ativado manualmente via ícone de lupa para uma experiência de compra superior.
- **Indicador de Stock Dinâmico**: Visualização clara de disponibilidade (Verde - Em Stock / Cinza - Sem Stock).
- **Galeria de Imagens**: Navegação fluida entre diferentes ângulos dos modelos 3D.
- **Carregamento Instantâneo**: Otimização total de imagens e uso do Next.js Turbo para navegação sem esperas.
- **Carrinho de Compras**: Gestão completa de pedidos com persistência de dados.

## 🛠️ Instalação Local

Para rodar este projeto no seu computador, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/danilohenriquesilvalira/PRINT3D.git
   cd PRINT3D
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Aceda no navegador:**
   [http://localhost:3000](http://localhost:3000)

## 📱 Acesso Remoto

O projeto está configurado para permitir acesso via rede local. Pode testar no seu telemóvel acedendo ao IP do seu computador na porta 3000.
